# Etch-a-Sketch

A browser-based version of the classic Etch-a-Sketch toy. This project focuses on **DOM manipulation**, dynamic CSS styling, and handling user input to create a fun, interactive drawing canvas.

## ✨ Features
* **Hover-to-Draw:** Tracks mouse movement to "color" the pixels.
* **Clear Canvas:** A reset button to wipe the board clean.
* **Rainbow Mode (Optional):** Generates random colors for every pixel touched.
* **Progressive Darkening:** Adds 10% black to the pixel with every pass.

## Tech Stack
* **HTML5:** Semantic structure for the UI and canvas container.
* **CSS3:** Flexbox/Grid for the layout and responsive design.
* **JavaScript (Vanilla):** Logic for grid generation and event listeners.

## Learning Objectives
This project was built to master the following concepts:
1.  **Creating Elements Dynamically:** Using `document.createElement()` and `appendChild()`.
2.  **Event Listeners:** Specifically `mouseenter` and `mousedown` for drawing logic.
3.  **CSS Variables:** Using JS to update CSS variables for grid dimensions.
4.  **UI/UX Basics:** Adding "Active" states to buttons and handling user alerts for grid resizing.
